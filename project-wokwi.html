Wokwi (IoT)

ทดลองต่อบอร์ดอาดูโนนผ่านเว็บ Wokwi

ตัวอย่างโปรเจกต์

องค์ประกอบในการทำโปรเจค

เว็บ woki

ระบบการทำงาน

การสร้างบอร์ดและโปรเจกต์ เราสามารถเลือกบอร์ด Arduino หรือ ESP32, ESP8266, หรือบอร์ดอื่น ๆ สามารถลากวางอุปกรณ์เสริม เช่น LED, ปุ่มกด, เซ็นเซอร์, มอเตอร์ ลงบนบอร์ดได้ ตั้งค่าขาของอุปกรณ์ให้ตรงกับโค้ด

ปัญหา และ วัตถุประสงค์

Wokwi ช่วยให้การทดลองโค้ดและวงจร Arduino เป็นเรื่องง่าย ปลอดภัย และเข้าถึงได้ทุกคน โดยไม่ต้องใช้บอร์ดจริง

วิธีแก้ และ แนวทาง

ใช้ แพลตฟอร์มออนไลน์ Wokwi เพื่อจำลองบอร์ด Arduino/ESP และอุปกรณ์ต่างได้ตามต้องการแล้วก็เหมาะสำหรับมือใหม่ที่อาจทำให้บอร์ดไหม้ได้ง่ายเพราะไม่รู้กจัการจ่ายไฟที่แต่ละอุปกรณ์กำหนด

ฟีเจอร์หลัก

  • Serial Monitor
  • ลากวางอุปกรณ์เสริม
  • บจำลองบอร์ด Arduino/ESP ออนไลน์
  • Code Editor และ Simulation Real-Time

สิ่งที่เรียนรู้

ควบคุม LED และอ่านปุ่ม/เซ็นเซอร์บนบอร์ด Arduino/ESP, ใช้ PWM ปรับความสว่าง, สร้างเอฟเฟกต์ไฟวิ่ง, และตรวจสอบค่าแบบ Real-Time ผ่าน Serial Monitor
