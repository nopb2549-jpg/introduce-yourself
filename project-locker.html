ระบบการทำงาน

ผู้ใช้ เข้าสู่ระบบ เลือกบริการตู้ล็อกเกอร์ ก่อนที่ลูกค้าจะเลือกบริการ ระบบจะตรวจสอบฐานข้อมูล ถ้าไม่ว่างจะแจ้งเตือนกลับไปยังผู้ใช้ ถ้าว่างระบบจะอนุญาตให้จอง Esp จะรับข้อมูลจาก Server เพื่อให้ตู้ล็อกเกอร์เปิด แล้ว Update สถานะตู้ว่าใช้งานอยู่

ปัญหา และ วัตถุประสงค์

ผู้ใช้งานตู้ล็อกเกอร์ทั่วไปมักประสบปัญหาการจัดการกุญแจ เช่น กุญแจหาย การใช้งานไม่สะดวก และมีความเสี่ยงด้านความปลอดภัย ระบบนี้ถูกพัฒนาขึ้นเพื่อปรับปรุงประสบการณ์ใช้งานให้ดีขึ้น โดยเปลี่ยนจากกุญแจแบบเดิมมาเป็น ระบบควบคุมตู้แบบออนไลน์ ให้ผู้ใช้สามารถเปิด–ปิดตู้ได้ผ่านเว็บไซต์อย่างปลอดภัยและรวดเร็ว

วิธีแก้ และ แนวทาง

เปิด–ปิดตู้ผ่านเว็บแอปแล้วสามารถระบบแสดงสถานะตู้แบบ Real-time

ฟีเจอร์หลัก

  • บอกเวลการจองที่เหลือของลูกค้า
  • คำนวนราคาการจองตู้ล็อกเกอร์
  • ควบคุมการเปิดปิดตู้ผ่านเว็บแอพ
  • แสดงสถานะตู้ล็อกเกอร์และเวลาคงเหลือในการใช้งานแบบ Real-time เพื่อแจ้งให้ผู้ใช้ทราบได้ล่วงหน้าว่าตู้ใดจะว่างเมื่อไหร

สิ่งที่เรียนรู้

เรียนรู้การเชื่อมต่อ Sensor กับ Arduino, การทำ Dashboard แบบเรียลไทม์ และการจัดการระบบ IoT ขนาดเล็ก
